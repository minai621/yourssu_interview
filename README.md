# 유어슈 회원목록 관리

## 실행 방법
```
yarn
yarn start
```
>http://localhost:3000 접속하기.
- 파일을 변경하면 자동 새로고침 됩니다.

## 요구 사항 
- 각각의 API는 일정 확률로 실패할 수 있음. 실패했을 경우 `alert` 메시지 상자 표시.
- 처음 들어가면 유저 불러오는 API를 통해 모든 유저들을 불러와 표에 표시하기.
- 이름, 닉네임, 이메일을 입력하고 등록을 누르면 등록하는 API를 통해 등록하고 유저 리스트를 갱신하기
- 새로고침 버튼을 누르면 API를 이용해서 유저 리스트 다시 불러오기
- 삭제 버튼을 누르면 해당 유저 탈퇴시키는 API 이용해서 탈퇴시키고 유저 리스트를 갱신하기
